OKShadow by OKFocus

OKShadow is a JQuery plugin that makes it easy to create drop shadows that react to mouse proximity. A variety of effects are possible, including depth-of-field simulation and focus effects.

Examples

A basic box shadow, used on the square at top left: 

$('#ok').okshadow({
  color: 'red'
});

A text shadow, used on the heading at the very top. xMax and yMax constrain the position of the shadow. fuzzMin and fuzzMax constrain the amount of blur, keeping the shadow blurry, but never completely in focus: 

$('h1').okshadow({
  color: '#bbb',
  textShadow: true,
  xMax: 10,
  yMax: 10,
  fuzzMin: 3,
  fuzzMax: 10,
});

A text shadow, used on the links at left. The transparent parameter sets the text-color to transparent, producing a ghost text effect. With both xMax and yMax set to zero, the shadow does not move. Additionally, CSS is used to make the text come sharply into focus on hover, while a fuzzMin setting keeps the other text blurry when the mouse is near: 

$('li a').okshadow({
  color: 'blue',
  textShadow: true,
  transparent: true,
  xMax: 0,
  yMax: 0,
  fuzzMin: 1.5
});
css:

a:hover {
  color: red !important;
  text-shadow: none !important;
}

Usage

color sets shadow color red
textShadow applies the shadow to the text; otherwise, the shadow affects the container ("box-shadow") false
transparent sets text color to transparent for a ghost text effect false
xMax constrains shadow's movement horizontally null
yMax constrains shadow's movement vertically null
xFactor ratio affecting blur position based on mouse distance 30
yFactor ratio affecting blur position based on mouse distance 30
fuzzMin lower bound for blur effect null
fuzzMax upper bound for blur effect null
fuzzFactor ratio affecting blur intensity to mouse distance.
See limits for a demonstration of these ratios		 40
